BHUK,LP TGBNHGYT BHUK,LP UYGBN TGBNHGYT BHUK,LP BHUK,LP TGBNHGYT BHUK,LP TGBNHGYT UYGBN

只是有一个分割的问题，一开始用,去分割就没有做出来，要用空格来分割，可以得到

```

TGBNHGYT

UYGBN 

TGBNHGYT 

BHUK,LP 

BHUK,LP 

TGBNHGYT 

BHUK,LP 

TGBNHGYT 

UYGBN

```

一共有3类

```

BHUK,LP

UYGBN

TGBNHGYT

```

![1076458764](http://ogmho3r7t.bkt.clouddn.com/2017-04-17-1076458764.jpg)

